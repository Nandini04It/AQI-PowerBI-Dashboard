# Air Quality Index Analysis Dashboard

This project analyzes air pollution levels across cities using Power BI.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Key Insights
- Identifies most polluted cities
- Shows AQI trends over time
- Compares pollutant levels
